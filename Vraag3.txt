Het fundamentele verschil tussen REST en SOAP is de manier hoe dat ze een server gaan aanspreken. Bij REST kiest men voor een logische opbouw in de URL zodat elk element in een db via een URL aangeproken kan worden mbv http protocols. Bij een SOAP service kiest men voor een achterliggende logica die alle protocols gaat uitvoeren in xml waarbij elk element van de db niet appart aanspreekbaar is.
